# Ejecutar la Aplicación localmente (Windows):

## Crear entorno virtual
```
py -m venv myenv

```
## Activar entorno virtual
```
myenv/Scripts/activate

```

## Instalar librerías
```
pip install -r requirements.txt

```

## Aplicación Grafo
```
py -m graph

```

## Aplicación Digrafo
```
py -m digraph

```
